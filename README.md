## Run Locally

Clone the project

```bash
  https://github.com/mannar-1/college-managemt-system.git
```

Go to the project directory

we start both frontend and backend server

#to start Backend
--------------------------------------------
```bash
  cd backend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
#to start frontend
--------------------------------------------
```bash
  cd frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
npm start
```
